# AI & ML Jobs Report Power BI Dashboard

This project is a comprehensive **Power BI Dashboard** that provides deep insights into the AI and Machine Learning job market. It analyzes various aspects such as job roles, salaries, experience levels, company hiring trends, locations, and in-demand skills.

## 📊 Dataset Overview

The dataset (`DATA/` directory) is structured relationally using the following tables:
- **Jobs**: Core information including Job Title, Experience Level, Employment Type, Remote Ratio, Salary (USD), and Posting Date.
- **Companies**: Details about companies hiring AI/ML professionals.
- **Locations**: Geographical data regarding job postings.
- **Skills**: A master list of relevant AI/ML skills.
- **Job_Skills**: A mapping table linking jobs to their required skills.

## 🚀 Dashboard Features & Insights

The dashboard is divided into several analytical views to provide a holistic understanding of the market.

### 1. Overview
A high-level summary of the AI & ML job landscape, highlighting key metrics such as total jobs, average salaries, and overall trends.

### 2. Company & Role Insights
An analysis of which companies are actively hiring, the types of roles they are looking for, and how compensation varies across different job titles and experience levels.

### 3. Location Insights
Geographical breakdown of job opportunities, comparing tech hubs, remote work ratios, and regional salary differences.

### 4. Skills & Hiring Insights
A deep dive into the most sought-after skills, the correlation between specific technical skills and salaries, and current hiring patterns.
